Detailed Description

In the dataframe, each row will be marked with one op these values:
    - 'arrow': draw a solid arrow
    - 'dot': draw a dot only
    - 'marked': marked, not to be drawn
    - 'pending': pending
    - 'adv': for reporting logical time advancing, draw a simple dash
