Detailed Description

The first element of every self struct defined in generated code will be a pointer to an allocation record, which is either NULL or the head of a NULL-terminated linked list of allocation records. Casting the self struct to this type enables access to this list by the function _lf_free_reactor(self_base_t*). To allocate memory for the reactor that will be freed by that function, allocate the memory using _lf_allocate(size_t,size_t,self_base_t*).
